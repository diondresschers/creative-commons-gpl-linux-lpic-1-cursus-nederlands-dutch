yum

  • YUM (Yellowdog Update Manager nu YUM Update Manager) is de software management tool van Red Hat gebaseerde distributies.
  • Onder water installeert YUM eigenlijk gewoon RPM (RedHat Package Manager) packages.
  • YUM installeert wel dependancies mee, waar RPM alleen maar waarschuwt.
LPIC-1 | Section 102.5 | Page 3/28

yum

  • Zoeken naar mogelijke programma's doe je met 'yum seach'.
  • Met 'yum install' kan je programma's installeren.
  • Verwijderen doe je met 'yum remove'.
  • De optie '-y' geeft op elke vraag tijdens de installatie het antwoord 'yes'

/etc/yum.conf

  • De configuatie van YUM staat in het bestand '/etc/yum.conf'.
  • Hierin kan je bv. de GPG security aan of uit zetten.
  • In deze file wordt weer verwezen waar de repositories staan.
  • Ook wordt er verwezen naar de YUM logfile.

rpm

  • RPM is de RedHat Package Manager
  • Je kan individele packages installeren met de optie '-i' RPM.
  • Je kan zoeken naar packages met '-q' (query) en '-a' (all).
  • Je kan packages weer verwijderen met '-e' (erase)
  • RPM zal standaard geen packages verwijderen die nog een depency hebben.
  • Wil je dit echter wel gebruikt dan de optie 'nodeps', met het risico programma's te breken.

yumdownloader

  • Met 'yumdownloader' kan je individuele 'rpm' files downloaden
  • Het zit niet standaard bij YUM, maar je het is onderdeel van 'yum-utils'
  • Je hebt dus alleen de 'rpm'-file, er is echter geen programma geinstalleerd.

rpm2cpio

  • Je kan rpm zijn packages, dus je kan ze ook uitpakken naar een 'cpio' file (een soort zip-file).
  • Dit doe je met het commando 'rpm2cpio'.
  • Het commando 'cpio' zelf valt buiten het examen, maarmee hier kan je van een 'cpio'-file weer individele files maken
  • Je kan de inhoudt van een 'cpio' file bekijken met 'less'

  • Get the source code in a tarball and install from that soource.
  • Get a source RPM and use rpm to install it.
  • Use rpm to install the program's binaries from an rpm packages.
  • Use cpio to extract the binaries from a source RPM and use rpm to install those sources.
