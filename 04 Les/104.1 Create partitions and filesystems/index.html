fdisk

  • De 'fdisk' tool is naast een commando een menu gestructureerd programma om partities te maken en wijzigen.
  • Alleen 'fdisk -l' laat een lijst met partities zien (vergeet de 'root' rechten niet).
  • Wees voorzichtig met fdisk, want voor je het weet verniel je je harde schijf partities (hint: gebruik een VM of USB stick).

LPIC-1 | Section SECTIE | Page 4/18

fdisk

  • a - toggle a bootable flag
    b - edit bsd disklabel
    c - toggle the dos compatibility flag
    d - delete a partition
    l - list known partition types
    m - print this menu
    n - add a new partition
    o - create a new empty DOS partition table
LPIC-1 | Section SECTIE | Page 5/18

fdisk

  • p - print the partition table
    q - quit without saving changes
    s - create a new empty Sun disklabel
    t - change a partition's system id
    u - change display/entry units
    v - verify the partition table
    w - write table to disk and exit
    x - extra functionality (experts only)

mkfs

  • Je zet een file system op een partitie met 'mkfs' (van make filesystem).
  • Je geeft het type van het soort filesyteem aan met de optie '-t'.
LPIC-1 | Section SECTIE | Page 13/18

mkfs

  • De volgende filesystemen zijn van belang:
  • ext2 het Linux filesysteem
  • ext3 het Linux filesysteem met journalling
  • ext4 is ext3 met grotere limieten
  • vfat voor Windows FAT
  • reiserfswaarvan de ontwikkeling zo goed als stil staat.
  • xfs extended filesystem van Silicon Graphics
