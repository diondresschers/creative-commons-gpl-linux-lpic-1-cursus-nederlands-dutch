ls

  • 'ls' laat een list zien van de directory
  • Optie '-l' laat een lange 'ls' zien met ook tijden, eigenaren en rechten op die file.
  • '-a' laat alle files zien, dus ook de verborgen (.) bestanden.
  • '-t' sorteert de 'ls' op basis van aanpas tijd.
  • '-r' sorteert reverse.

tar

  • 'tar' staat voor tape archive.
  • Alhoewel tapes niet veel gebruikt meer worden, is het een methode om meerdere files te backuppen (of te 'tarren').
  • Een 'tar' file is een soort file waar weer meedere files inzitten.
  • Via 'tar' files kan je niet alleen 'tar' files tarren en untarren, maar ook nog op verschillende manieren comprimeren.
  • De opties moeten wel in de goede volgorde staan...
LPIC-1 | Section 103.3 | Page 34/64

tar

  • -c creert een tar file
  • -x extraheert een tar file
  • -v (verbose) geeft veel informatie
  • -f defineer de getarde of de untarde file.
  • -z zorgt dat de tar file nog gezipt wordt.

cpio

  • Met 'cpio' (copy in out) kan je bestanden inpakken en uitpakken.
  • Je kan echter ook 'cpio' input geven van bv. het 'find' commando, waardoor je beter kan selecteren welke bestanden je wilt archiferen.
  • De optie '-o' (van out) maakt er een archief van.
  • Met '-H' kan je het type achrifering kiezen.
  • De '-t' van try laat zien wat voor bestanden er in het archief zitten.

Vraag 1

Bob accidentilly created the subdirectory \bobsdir in his home directory. He tried to remove the subdirectory with the command rmdir \bobsdir only to receive the error, "No such file or directory." Which command will remove the directory?


  • rmdir 'bob/\bobsdir'
  • rmdir "bob/\bobsdir"
  • rmdir ~bob/ 'bobsdir
  • rmdir bob/\bobsdir
  • rmdir ~bob/\\bobsdir

Vraag 2

John works as a Network Administrator for Perfect Solutions Inc. The company has a Linux-based network. John is working as a root user on the Linux operating system. He wants to change the modified data and time of the file private.txt to 11 Nov 2009 02:59:58 am. Which of the following commands will John use to accomplish his task? Each correct answer represents a complete solution. Choose all that apply.


  • touch -d "11 Nov 2009 02:59:58 am" private.txt
  • touch -t 200911110259.58 private.txt
  • rm private.txt #11 Nov 2009 02:59:58
  • touch private.txt #11 Nov 2009 02:59:58 am
