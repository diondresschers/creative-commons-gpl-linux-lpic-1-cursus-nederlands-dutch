quota

  • 'quota' oftewel gebruikers datalimieten staat niet standaard op een Linux desktop.
  • Je kan het installeren met 'apt-get install quota' of 'yum install quota'.
  • Je zet de quota aan op een filesysteem.
  • Met 'quota' kan je je eigen user quota zien.

quotaon

  • Je kan de quota aan zetten met 'quotaon'.
  • Geef wel het mountpunt op.
  • Quota kan je niet op een enkele directory zetten.
  • Je moet in de /etc/fstab bij de opties wel 'usrquota' (voor user quota) toevoegen.
  • Gebruik 'grpquota' voor de group quota.
  • Je krijgt daarbij de bestanden 'aquota.user' en/of 'aquota.group'.
  • Deze bestanden zijn geen ASCII bestanden maar binaries.
LPIC-1 | Section 104.4 | Page 6/18

quotaon

  • Je kan quota zien in een file formaat (met 'vi' of 'nano') waar je de o.a. de volgende instellingen ziet: 'blocks' 'soft' 'hard'
  • Je ziet het aantal 'blocks' dat de gebruiker in gebruik heeft.
  • Je kan een 'soft'-limiet instellen voor het aantal blocks (deze kan tijdelijk overschreven worden), na een week (default) mag er geen data meer bij geplaatst worden.
  • Je ziet een 'hard' limiet voor de blocks, hier mag niet overeen gegaan worden.
LPIC-1 | Section 104.4 | Page 7/18

quotaon

  • Er staan ook de volgende inode instellingen (dus eigenlijk aantal files/directories):
  • 'inodes' 'soft' 'hard'
  • Het aantal 'inodes' dat de user gebruikt.
  • De 'soft' limiet van 'inodes' deze mag je weer overschrijven voor een week, je krijgt ook (weer) een notificatie.
  • De 'hard' limiet mag je dus weer niet overschrijven.

edquota

  • Met 'edquota' (edit quota) kan je quota's gaan zetten.
  • Geef de optie '-u' mee om de desbetreffende user (gebruiker) te selecteren.
  • Gebruik '-g' voor de desbetreffende group (group).
