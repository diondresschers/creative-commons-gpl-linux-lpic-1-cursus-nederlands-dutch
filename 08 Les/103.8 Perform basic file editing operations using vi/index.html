Wat is VI?

  • VI (visual interface) is de meest gebruikte teksteditor van Unix-sytemen.
  • VI is ontstaan in de jaren 70 en is ook standaard geinstalleerd op (bijna) alle Linux distrubuties.
  • Doordat VI als lastig en niet gebruiksvriendelijk werd gezien is de editor VIM (VI iMproved) ontstaan.
  • De meeste distributies gebruiken VIM en niet VI. Om het wat duidelijker te maken: als je de opdracht 'vi' typt, wordt eigenlijk 'vim' gestart.
  • Echter wordt VIM ook nog als moeilijk bestempeld en sommige distrubuties gebruiken daarom 'nano'.
LPIC-1 | Section 103.8 | Page 3/12

Wat kan je met VI?

  • Met de teksteditor VI kan je visueel alle taken uitvoeren die je zou kunnen verwachten van een teksteditor.
  • Taken zoals nummering, zoeken, wijzigen, opslaan zijn uit te voeren met VI.
  • Via VIM (modules) kan je zelf kleuren weergeven en autocompletion bewerkstelligen.
  • In VI werk je in twee modes: de command mode en de insert mode. Standaard zit je in de command mode.
  • Meer informatie vind je op de wikipedia pagina van VI of VIM

Begrippen en commando's

  • 'esc' hiermee kan je terug naar de command mode
  • 'i' hiermee ga je naar de insert (invoer) mode op de huidige positie
  • '/' hiermee kan je vooruit zoeken vanaf de huidige cursor positie en met de letter 'n' kan je naar de volgende
  • '?' hiermee kan je terug zoeken vanaf de huidige cursor positie
  • 'h, j, k, l' worden gebruikt als pijltoetesen
    h = links l = rechts j = beneden k = boven
  • 'o' hiermee kan je een regel openen onder de positie van de cursor
  • 'O' hiermee kan je een regel openen boven de positie van de cursor
LPIC-1 | Section 103.8 | Page 5/12

Begrippen en commando's

  • 'yy' hiermee kan je een regel copieren (y staat voor yank)
  • '5yy' hiermee kan je 5 regels copieren (y staat voor yank)
  • 'p' hiermee kan je plakken wat in de buffer staat
  • 'dd' hiermee delete je een regel
  • '18dd' hiermee delete je 18 regels
LPIC-1 | Section 103.8 | Page 6/12

Begrippen en commando's

  • ':q' hiermee sluit je VI af zonder op te slaan.
  • ':w' hiermee schrijf je de wijzigingen weg ('w' van 'w'rite)
  • '!' hiermee kan je een commando forceren (bv 'q!')
  • ':wq!' Dit is een combinatie tussen 'w'rite en 'q'uit
  • ':set number' Laat regelnummers zien.
  • ':syntax on' Gebruik kleuren (of gebruik 'syntax off' om het uit te zetten).
  • ':s/OLD/NEW/g' Gebruik de stream editor in VI.
  • '!ls' Geef het bash commando 'ls'
  • Typ 'vmtutor' in de CLI voor een hands-on oefening!
