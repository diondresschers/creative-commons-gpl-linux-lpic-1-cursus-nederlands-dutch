Braille Display
Een Braille Display is een stukje hardware waarmee blinden tekst op het scherm via de braille tekens kunnen lezen.
<imga>

High Contrast Desktop Themes
Je kan het uiterlijk van de desktop veranderen zo dat er meer contrast zichtbaar is. De knoppen, randen en eventueel iconen worden duidelijker weergegeven.

Large Print Desktop Themes
Je kan ervoor kiezen dat het standaard lettertype groter wordt weergegeven. Zo wordt de tekst duidelijker weergegeven.

Sticky Keys
ipv bv ALT+F2, druk eerst ALT dan pas F2

Slow Keys
Een korte aanslag (bv. een perongeluke aanslag) van een letter doet niks, als je de toets langer indrukt wordt de aanslag geaccepteerd.

Bounce Keys
Het negeren van dubbele aanslagen (bv twee keer een 'a' wordt geregistereerd als 'a'

Mouse Keys
Als je een nummeriek gedeelte hebt op je toetsenbord kan je de toetsen gebruiken om de muis te besturen (bv 4 is links en 8 is naar boven).
