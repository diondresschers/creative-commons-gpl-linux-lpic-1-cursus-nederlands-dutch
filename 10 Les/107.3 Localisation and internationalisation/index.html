/etc/timezone

  • De file '/etc/timezone' laat zien wat de default tijdzone is van het systeem
  • Deze file is wel te vinden onder Ubuntu, maar niet onder RedHat
  • Normaal gesproken is dit een file van een regel waar de tijdzone staat (bv. 'Africa/Morrocco')

/etc/localtime

  • De file '/etc/localtime' is een symbolic link naar de huidige timezonefile in de '/usr/share/zoneinfo' directory
  • In '/usr/share/zoneinfo' zitten alle tijdzone files.
  • Deze timezone file (bv. 'America/New York') is zelf een binary file

/usr/share/zoneinfo

  • In '/usr/share/zoneinfo' staan alle binary zone files
  • In Ubuntu is de '/etc/localtime' een copy van de binary zone file.
  • In RedHat varianten is dit geen copy maar wordt gebruik gemaakt van een symbolic link.

/usr/bin/locale

  • Elk systeem heeft locale taalinstellingen van geld, tot tijd van adres tot telefoonnummer.
  • Je kan deze taaltinstellingen vinden door het commando 'locale'
  • Hierin staan verschillende instellingen die je kan overrulen door een individuele environment variabele te zetten bv, 'LC_TIME="en_US.utf8"'
  • De environment variabele 'LC_ALL' overruled alle losse 'LC_' variabelen weer.

date

  • Met 'date' kan je de huidige datum en tijd weergeven.
  • Je kan met 'date' echter ook de tijd/datum wijzigen (via 'sudo').
  • De tijd en datum wijzig je met 'date MMDDhhmm'.
  • Optioneel kan je ook nog het jaartal meegeven: 'date MMDDhhmmYYYY'.

UTF-8 & ISO-8859 & ASCII & Unicode

  • Om de juiste letters en karakters van het een naar het andere systeem of applicatie te sturen moet er een protocol zijn zodat alle karakters (ook de speciale) correct worden weergegeven.
  • Vroeger kon je met minder karaters af, maar tegenoordig zijn er meer en meer karakters, talen, en symbolen bijgekomen.
  • ASCII (encodering met slechts 128 karakters)
  • ISO-8859 (encodering en meer vreemde tekens)
  • unicode(de karakterset met de meeste tekens)
  • UTF-8 (de encodering van Unicode)
LPIC-1 | Section 107.3 | Page 27/32

iconv

  • Met 'iconv' kan je encodering formaten omzetten
  • Met 'iconv -l' zie je de hele waslijst met ondersteunde formaten.
  • Dit kan handig zijn op systemen die bepaalde formaten niet ondersteunen.
  • De standaard ouput is naar het scherm.
  • Voorbeeld: 'iconv -f utf-8 -t ascii bestand' (de 'f' van from de 't' van to)

Vraag 1

Which command will set the local machine's timezone to UTC?


  • cat UTC > /etc/timezone
  • ln -s /usr/share/zoneinfo/UTC /etc/localtime
  • date -timezone=UTC
  • mv /usr/timezone/UTC /etc
