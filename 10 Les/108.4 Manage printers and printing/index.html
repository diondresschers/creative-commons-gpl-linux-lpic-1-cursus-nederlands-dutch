CUPS

  • In 1982 werd PostScript uitgevonden. Een taal waarin je pagina's naar een (Laser) printer kan uitvoeren. Printers moesten PostScript kunnen verwerken.
  • Tegenwoordig kennen de meeste huis-tuin en keukenprinters geen PostScript printers meer
  • Het programma GhostScript vertaalt/rendert PostScript zodat de printer de pagina kan uitprinten.
  • PostScript is inmiddels ook vervangen door PDF (Printable Document Format).
  • GhostScript kan ook gebruikt worden als vertaler van PostScript naar PDF.
LPIC-1 | Section 108.4 | Page 3/19

CUPS

  • Linux werkt met een zogenaamde 'print queue'; een rij van print opdrachten
  • Deze print queue wordt nu bijgehouden door het programma CUPS. De linux computer wordt daardoor een printserver (al dan niet lokaal).
  • CUPS (Common Unix Printing System) is standaard geinstalleerd en is zelfs te benaderen in een GUI (via http://localhost:631).
  • Log in met de 'root' (niet met je eigen account).
  • Je kan via die GUI ook op verschillende manieren printers toevoegen (via de 'Administration'-tab).
  • Werk je op een CLI, dan kan je gebruik maken van een CLI browser zoals 'lynx' of 'links'

/etc/cups

  • De directory van cups is '/etc/cups'
  • De belangrijkste files zijn daar de 'cupsd.conf' (de CUPS configuratie) en de 'printers.conf' (de geregistreerde printers)
  • In 'cupsd.conf' verander je 'Listen localhost:631' in 'Listen :631' als je ook andere printers toegang wilt geven tot CUPS
  • In 'cupds.conf' kan je ook op basis van IP adressen, computers in een Apache-achtige (webserver) manier toegang geven tot bepaalde print-queues.
  • 'printers.conf' kan je instellen via de CLI, maar wellicht beter via de GUI.

lpd legacy

  • De printopdrachten verschijnen in '/etc/spool/cups'
  • We gebruiken nu CUPS, maar nog steeds werken de oude (legacy) commando's van 'lpd'
  • 'lpq' laat de print queue zien
  • 'lpq -a' laat alle print queues zien
  • 'lpq -Pprinter_naam' geldt voor die specifieke printer (geldt ook voor andere 'lpd' commando's.)
LPIC-1 | Section 108.4 | Page 12/19

lpd legacy

  • 'lpr README' zend het bestand 'README' naar de (default) printer
  • 'lpr -PHP_DeskJet520 README' zend het bestand 'README' naar de printer: 'HP_DeskJet520'
  • 'lprm 32' verwijdert print job nummer '32' (vind de job nummers met 'lpq -a')
  • 'sudo lprm -' verwijdert alle print jobs

Vraag 1

You need to pause the CUPS printer HPLaserjet4, and you want to cancel all print jobs with a message, "hello". Which command will do this?


  • cupsreject -c -r hello HPLaserjet4
  • cupsreject -p -m hello HPLaserjet4
  • cupsdisable -c -r hello HPLaserjet4
  • cupsdisable -p -m hello HPLaserjet4
