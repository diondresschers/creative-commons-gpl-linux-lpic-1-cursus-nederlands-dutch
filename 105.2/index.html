for

  • Bij het schrijven of aanpassen van scripts kan je gebruik maken van een zogenaamde loop (een herhaling).
  • Deze loop kan in de vorm zijn van een 'for'-loop.
  • Hiermee herhaal je een bepaalde actie een aantal keer.
  • Probeer bij het scripten zoals in volgend voorbeeld, statische zaken zoals 'dion' altijd te vermijden. Hierdoor werkt je programma ook voor andere gebruikers en op andere computers.
LPIC-1 | Section 105.2 | Page 3/22

for

#!/bin/bash
for FILE in `ls /home/dion/*.JPG`
  do
  firefox $FILE
done

while

  • Een andere vorm van een loop is de 'while' loop.
  • Hiermee doe je iets zolang een bepaalde conditie geldt.
  • Spaties aan het begin van bash-scripts zijn optioneel, ze worden gebruikt om de code wat leesbaarder te maken.
LPIC-1 | Section 105.2 | Page 6/22

while

#!/bin/bash
NUMMER=1
while (($NUMMER <= 1005 ))
  do
  echo "gebruiker met $NUMMER:"
  id $NUMMER
  NUMMER=$(( NUMMER+1 ))
done

if

  • Je kan met een 'if' controleren of iets waar is of niet.
  • Bijvoorbeeld: is het vandaag een oneven dag of heeft de gebruiker het goede passwoord ingegeven.
  • Je kan een 'if' uitbreiden met een 'else', dus als het bovenstaande niet geldt, dan doe het volgende.
  • Je kan zelfs tussen de 'if' en 'else', meerdere andere opties opnemen onder 'elfseif'.
  • De 'if'-loop wordt afgesloten met een 'fi' (dat is 'if' achterstevoren)
  • Bovenaan het script zie je '#!/bin/bash', dit is geen commentaar, maar de scripttaal die gebruikt moet worden (bash in dit geval).

if

#!/bin/bash
echo $USERNAME
echo "Moet je home dir weg?"
read ANTWOORD
if [ $ANTWOORD = "Jahaa!" ];
then
  rm -rf /home/$USERNAME
else
  touch /home/$USERNAME/nietweg
fi

test

  • Je kan op basis van 'test' vele zaken testen

seq

  • Met 'seq' kan een je sequentie van getallen uitvoeren.
  • 'seq 5' (5=eind) resulteert in:1, 2, 3, 4, 5
  • 'seq 20 26' (20=begin, 26=eind) resulteert in: 20, 21, 22, 23, 24, 25, 26
  • 'seq 0 100 500' (0=begin, 100=stap 500=eind) resulteert in: 0, 100, 200, 300, 400, 500
